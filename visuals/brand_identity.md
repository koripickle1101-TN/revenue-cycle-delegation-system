# Brand Identity

## Project Brand

The Revenue Cycle Delegation System

## Brand Positioning

Premium healthcare administration portfolio system focused on workflow visibility, revenue cycle delegation, upstream denial prevention, and AI-supported administrative intelligence.

## Core Message

Stop prompting. Start delegating healthcare workflow intelligence.

---

## Color System

| Color Role | Hex | Use |
|---|---|---|
| Primary Background | `#FFFFFF` | Main page and carousel background |
| Primary Text | `#111111` | Headlines and important copy |
| Secondary Text | `#333333` | Body text and supporting explanations |
| Tennessee Orange | `#FF8200` | Accent lines, numbers, nodes, key phrases, dividers |
| Warm Gray | `#F6F3EF` | Soft cards, section backgrounds, subtle structure |
| Light Border Gray | `#E7E1DA` | Thin lines, card borders, workflow connectors |

---

## Typography Direction

| Type | Style |
|---|---|
| Headlines | High-contrast editorial serif, large scale, confident spacing |
| Body Copy | Clean sans-serif, readable, calm, professional |
| Labels | Small caps, wide letter spacing, Tennessee Orange accents |
| Signature | Elegant handwritten-style cursive when used in visuals |

---

## Visual System

Use:

- High whitespace
- Editorial asymmetry
- Thin divider lines
- Double-ring circular workflow nodes
- Dotted connector lines
- Soft warm gray structure blocks
- Minimal healthcare operations iconography
- Black text with Tennessee Orange emphasis
- Clean portfolio-ready layout

Avoid:

- Heavy gradients
- Overcrowded slides
- Generic medical stock imagery
- Clipart-style icons
- Neon overload in this specific project
- Clinical claims or patient-treatment language
